# simple_ocr
Simple OCR for tessecart on python 3.6

Программа распознавания текстов с изображения.

Поддержка языков:
- Английский
- РУсский
- Японский
- Китайский традиционный
- Китайский упрощенный

Требования:
* Установленный tesseract
* Python 3.6
* Модули
	- PyQt5
	- PIL
	- pytesseract
